# 语音合成API文档
## ZmeetAI 语音合成优势

* 多达数千种模特语音发音模型，每种模型分别带有不同的感情音色风格
* 极速版本可在100毫秒以内返回语音，支持感情定制
* 轻便版本感情细腻，发声细腻
